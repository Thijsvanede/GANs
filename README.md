# GANs
Implementation of various Generative Adversarial Networks (GANs) in Keras.

## Installation
The repository can be cloned using
```bash
git clone https://github.com/Thijsvanede/GANs.git
```

```bash
GANs

├── LICENSE
├── README.md
├── saved
└── src
    └── GAN.py
```
